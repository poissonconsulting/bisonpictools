# Load analysis

Loads analysis object from file path where it has been saved.

## Usage

``` r
bpt_load_analysis(file)
```

## Arguments

- file:

  Path to file where analysis object is saved (excluding file
  extension).

## Value

An smb_analysis object.

## Examples

``` r
if (FALSE) { # \dontrun{
bpt_load_analysis("file_path/analysis")
} # }
```
