# Sample data for calf proportion data

A data set of invented calf proportion data.

## Usage

``` r
proportion_calf_data
```

## Format

A tibble with 2 rows and 5 columns:

- proportion_calf_year:

  The year of the calf proportion survey

- proportion_calf_month:

  The month of the calf proportion survey

- proportion_calf_day:

  The day of the calf proportion survey

- proportion_calf:

  Estimated calf proportion

- proportion_calf_cv:

  Coefficient of variation (standard deviation / estimate) of calf
  proportion estimate
