# Save analysis object

Saves analysis object to the file location specified by `file`.

## Usage

``` r
bpt_save_analysis(analysis, file)
```

## Arguments

- analysis:

  Analysis object from
  [`bpt_analyse()`](https://poissonconsulting.github.io/bisonpictools/reference/bpt_analyse.md).

- file:

  File path to save analysis object to (file extension added
  automatically).

## Examples

``` r
if (FALSE) { # \dontrun{
bpt_save_analysis(analysis, "output/analysis")
} # }
```
